# iOS PWAアイコン更新クイックガイド

## 🚀 最短手順（3ステップ）

### 1. ホーム画面から削除
- アプリアイコンを**長押し** → **「Appを削除」**をタップ

### 2. Safariのキャッシュをクリア
- **設定** → **Safari** → **「履歴とWebサイトデータを消去」**

### 3. ホーム画面に再追加
- SafariでPWAを開く → **共有ボタン** → **「ホーム画面に追加」**

## ✅ これで完了！

新しいアイコンがホーム画面に表示されます。

---

## 📝 詳細手順（分からない場合）

### ステップ1: アプリアイコンを削除

1. ホーム画面でアプリアイコンを**長押し**します
2. アイコンが揺れ始めます（編集モード）
3. アイコンの左上に「×」ボタンが表示されます
4. 「×」をタップ、または「Appを削除」を選択します

**注意**: アプリのデータは削除されません！

### ステップ2: Safariのキャッシュをクリア

1. iPhoneの「**設定**」アプリを開きます
2. スクロールして「**Safari**」をタップします
3. さらに下にスクロールして「**履歴とWebサイトデータを消去**」をタップします
4. 確認ダイアログで「**履歴とデータを消去**」をタップします

### ステップ3: PWAを再追加

1. **Safari**を起動します
2. PWAのURLを開きます（ブックマークから、またはURLを直接入力）
3. 画面下部の**共有ボタン**（四角から矢印が出ているアイコン）をタップします
4. メニューをスクロールして「**ホーム画面に追加**」をタップします
5. アイコン名を確認して「**追加**」をタップします
6. ホーム画面で新しいアイコンが表示されていることを確認します

## 🔍 確認方法

### アイコンが正しく更新されているか確認

1. **ホーム画面で確認**
   - 新しいクレジットカードアイコンが表示されているか

2. **Safariで確認**
   - PWAを開いたときに、新しいアイコンが表示されているか
   - 共有ボタン → ホーム画面に追加 → アイコンが新しいデザインか

## ❓ トラブルシューティング

### まだ古いアイコンが表示される場合

1. **もう一度削除して再追加**
   - キャッシュが残っている可能性があります

2. **iPhoneを再起動**
   - システムのキャッシュをクリアできます

3. **Safariのデータを完全に消去**
   - 設定 → Safari → 「Webサイトデータを消去」
   - より強力なキャッシュクリアです

### アイコンが表示されない場合

1. **ファイルパスを確認**
   - `web/icons/apple-icon-180.png`が存在するか確認

2. **サーバーに正しくデプロイされているか確認**
   - アイコンファイルが正しく配信されているか

3. **HTMLの設定を確認**
   - `web/index.html`に`apple-touch-icon`が正しく設定されているか

## 💡 ヒント

- **開発中**: アイコンを変更するたびに削除→再追加が必要です
- **本番環境**: ユーザーにも同様の手順を案内する必要があります
- **バージョン管理**: アイコンを変更した場合は、バージョンを更新することを検討してください
